7.2 函数例子

以下是一些MaxCompute SQL支持的数学函数的示例：

  • ABS：计算绝对值。

    SELECT ABS(-10); -- 返回 10

  • ACOS：计算反余弦值。

    SELECT ACOS(0.5); -- 返回 1.0471975511965979

  • ATAN2：计算expr1/expr2的反正切值。

    SELECT ATAN2(1, 1); -- 返回 0.7853981633974483

  • CEIL：计算向上取整值。

    SELECT CEIL(3.14); -- 返回 4

  • COS：计算余弦值。

    SELECT COS(0); -- 返回 1

  • EXP：计算指数值。

    SELECT EXP(1); -- 返回 2.718281828459045

  • LOG：计算log对数值。

    SELECT LOG(10); -- 返回 2.302585092994046

  • POW：计算幂值。

    SELECT POW(2, 3); -- 返回 8

  • RAND：返回随机数。

    SELECT RAND(); -- 返回随机数值

  • SIN：计算正弦值。

    SELECT SIN(0); -- 返回 0
