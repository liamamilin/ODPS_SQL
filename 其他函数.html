8.1 窗口函数

函数 功能
AVG 对窗口中的数据求平均值。
CLUSTER_SAMPLE 用户随机抽样。返回True表示该行数据被抽中。
COUNT 计算窗口中的记录数。
CUME_DIST 计算累计分布。
DENSE_RANK 计算排名。排名是连续的。
FIRST_VALUE 取当前行所对应窗口的第一条数据的值。
LAG 取当前行往前（朝分区头部方向）第N行数据的值。
LAST_VALUE 取当前行所对应窗口的最后一条数据的值。
LEAD 取当前行往后（朝分区尾部方向）第N行数据的值。
MAX 计算窗口中的最大值。
MEDIAN 计算窗口中的中位数。
MIN 计算窗口中的最小值。
NTILE 将数据顺序切分成N等份，返回数据所在等份的编号（从1到N）。
NTH_VALUE 取当前行所对应窗口的第N条数据的值。
PERCENT_RANK 计算排名。输出百分比格式。
RANK 计算排名。排名可能不连续。
ROW_NUMBER 计算行号。从1开始递增。
STDDEV 计算总体标准差。是STDDEV_POP的别名。
STDDEV_SAMP 计算样本标准差。
SUM 对窗口中的数据求和。

以下是一些窗口函数的示例：

  • AVG：对窗口中的数据求平均值。

    SELECT AVG(salary) OVER (PARTITION BY department_id) FROM employee;

  • COUNT：计算窗口中的记录数。

    SELECT COUNT(*) OVER (PARTITION BY department_id) FROM employee;

  • MAX：计算窗口中的最大值。

    SELECT MAX(salary) OVER (PARTITION BY department_id) FROM employee;

  • ROW_NUMBER：计算行号。从1开始递增。

    SELECT ROW_NUMBER() OVER (ORDER BY salary DESC) AS row_num FROM employee;

  • SUM：对窗口中的数据求和。

    SELECT SUM(salary) OVER (PARTITION BY department_id) FROM employee;

以下是一些字符串函数的示例：

  • CONCAT：将字符串连接在一起。

    SELECT CONCAT('Hello', ' ', 'World'); -- 返回 'Hello World'

  • LENGTH：计算字符串的长度。

    SELECT LENGTH('MaxCompute'); -- 返回 11

  • SUBSTR：返回指定位置开始的子串。

    SELECT SUBSTR('MaxCompute', 4); -- 返回 'Compute'

  • UPPER：将字符串转换为大写形式。

    SELECT UPPER('maxcompute'); -- 返回 'MAXCOMPUTE'

  • ASCII：返回字符串的第一个字符的ASCII码。

    SELECT ASCII('A'); -- 返回 65

  • CHR：将指定ASCII码转换成字符。

    SELECT CHR(65); -- 返回 'A'

  • CONCAT_WS：将参数中的所有字符串按照指定的分隔符连接在一起。

    SELECT CONCAT_WS('-', '2022', '02', '14'); -- 返回 '2022-02-14'

  • LENGTHB：计算字符串以字节为单位的长度。

    SELECT LENGTHB('MaxCompute'); -- 返回字符串长度的字节数

  • LOCATE：在字符串中查找另一指定字符串的位置。

    SELECT LOCATE('abc', 'defabcghi'); -- 返回 4

  • REPLACE：将字符串中与指定字符串匹配的子串替换为另一字符串。

    SELECT REPLACE('hello world', 'world', 'MaxCompute'); -- 返回 'hello MaxCompute'

  • REVERSE：返回倒序字符串。

    SELECT REVERSE('MaxCompute'); -- 返回 'etupmoeCxaM'

  • SPLIT：按照分隔符分割字符串后返回数组。

    SELECT SPLIT('apple,banana,orange', ','); -- 返回数组 ['apple', 'banana', 'orange']

  • TOUPPER：将字符串中的英文字符转换为大写形式。

    SELECT TOUPPER('MaxCompute'); -- 返回 'MAXCOMPUTE'

  • TRIM：去除字符串的左右两端字符。

    SELECT TRIM('  MaxCompute  '); -- 返回 'MaxCompute'
